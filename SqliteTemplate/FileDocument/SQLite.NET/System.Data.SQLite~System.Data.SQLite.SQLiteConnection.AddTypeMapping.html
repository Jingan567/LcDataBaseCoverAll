SQLiteConnection.AddTypeMapping Method

Adds a per-connection type mapping, possibly replacing one or more that already exist.

public int AddTypeMapping(
   string typeName,
   DbType dataType,
   bool primary
);

Parameters

typeName
The case-insensitive database type name (e.g. "MYDATE"). The value of this parameter cannot be null. Using an empty string value (or a string value consisting entirely of whitespace) for this parameter is not recommended.
dataType
The DbType value that should be associated with the specified type name.
primary
Non-zero if this mapping should be considered to be the primary one for the specified DbType.

Return Value

A negative value if nothing was done. Zero if no per-connection type mappings were replaced (i.e. it was a pure add operation). More than zero if some per-connection type mappings were replaced.
