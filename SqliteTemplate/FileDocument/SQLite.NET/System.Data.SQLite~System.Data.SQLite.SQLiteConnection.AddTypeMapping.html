SQLiteConnection.AddTypeMapping 方法

添加每个连接类型映射，可能会替换一个或多个已存在的类型映射。

public int AddTypeMapping(
   string typeName,
   DbType dataType,
   bool primary
);

Parameters

typeName
不区分大小写的数据库类型名称（例如“MYDATE”）。此参数的值不能为 null。不建议对此参数使用空字符串值（或完全由空格组成的字符串值）。
dataType
应与指定类型名称关联的 DbType 值。
primary
如果应将此映射视为指定 DbType 的主映射，则为非零。

返回值

如果未执行任何作，则为负值。如果未替换每个连接类型的映射，则为零（即这是一个纯粹的 add作）。如果替换了某些每连接类型的映射，则大于零。
