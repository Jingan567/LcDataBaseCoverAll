ISQLiteManagedModule.RollbackTo 方法

调用此方法是为了响应 xRollbackTo 方法。

SQLiteErrorCode RollbackTo(
   SQLiteVirtualTable table,
   int savepoint
);

Parameters

table
与此虚拟表关联的 SQLiteVirtualTable 对象实例。
savepoint
这是一个整数标识符，用于为虚拟表指定特定的已保存状态，以便它将自身恢复到该状态，这也应该具有删除整数标识符大于此状态的所有已保存状态的效果。

返回值

标准 SQLite 返回代码。
