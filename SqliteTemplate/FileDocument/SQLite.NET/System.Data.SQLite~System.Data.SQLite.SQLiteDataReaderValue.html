SQLiteDataReaderValue Class

This class represents a single value to be returned from the SQLiteDataReader class via its GetBlob, GetBoolean, GetByte, GetBytes, GetChar, GetChars, GetDateTime, GetDecimal, GetDouble, GetFloat, GetGuid, GetInt16, GetInt32, GetInt64, GetString, or GetValue method. If the value of the associated public field of this class is null upon returning from the callback, the null value will only be used if the return type for the SQLiteDataReader method called is not a value type. If the value to be returned from the SQLiteDataReader method is unsuitable (e.g. null with a value type), an exception will be thrown.
