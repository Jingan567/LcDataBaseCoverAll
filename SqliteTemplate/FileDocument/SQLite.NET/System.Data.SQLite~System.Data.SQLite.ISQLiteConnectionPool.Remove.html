ISQLiteConnectionPool.Remove 方法

从与指定数据库文件名关联的连接池中删除连接，以便使用它与数据库进行交互。

   string fileName,
   int maxPoolSize,
   out int version
);

Parameters

fileName
数据库文件名。
maxPoolSize
指定数据库文件名的连接池的新最大大小。
version
与返回的数据库连接句柄关联的连接池版本（如果有）。

返回值

与指定的数据库文件名关联的数据库连接句柄，如果无法获取，则为 null。
