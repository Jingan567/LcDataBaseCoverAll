ISQLiteChangeSet Methods

The methods of the ISQLiteChangeSet interface are listed below. For a complete list of ISQLiteChangeSet interface members, see the ISQLiteChangeSet Members topic.

Public Instance Methods

ApplyOverloaded. Attempts to apply the set of changes in this instance to the associated database.
CombineWith This method combines the specified set of changes with the ones contained in this instance.
Invert This method "inverts" the set of changes within this instance. Applying an inverted set of changes to a database reverses the effects of applying the uninverted changes. Specifically: <ul><li> Each DELETE change is changed to an INSERT, and </li><li> Each INSERT change is changed to a DELETE, and </li><li> For each UPDATE change, the old.* and new.* values are exchanged. </li></ul> This method does not change the order in which changes appear within the set of changes. It merely reverses the sense of each individual change.
