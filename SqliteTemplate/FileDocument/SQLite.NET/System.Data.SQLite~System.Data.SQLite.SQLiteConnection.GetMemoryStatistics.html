SQLiteConnection.GetMemoryStatistics Method

Returns various global memory statistics for the SQLite core library via a dictionary of key/value pairs. Currently, only the "MemoryUsed" and "MemoryHighwater" keys are returned and they have values that correspond to the values that could be obtained via the MemoryUsed and MemoryHighwater connection properties.
