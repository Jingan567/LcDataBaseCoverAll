SQLiteBlob.Create(SQLiteDataReader, Int32, Boolean) Method

Creates a SQLiteBlob object. This will not work for tables that were created WITHOUT ROWID -OR- if the query does not include the "rowid" column or one of its aliases -OR- if the SQLiteDataReader was not created with the KeyInfo flag.
