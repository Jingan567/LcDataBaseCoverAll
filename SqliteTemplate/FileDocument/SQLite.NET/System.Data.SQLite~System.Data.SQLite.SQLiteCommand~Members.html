SQLiteCommand 概述

Public Static 字段

ForceExtraReads 此整数值与 CommandBehavior 值一起使用。设置后，将在 ExecuteScalar 方法中为所有事务执行额外的 Read 调用。应格外小心地使用此值，因为它可能会导致异常行为。
GlobalCommandBehaviors 这些是额外的命令行为标志，应用于对 ExecuteNonQuery、ExecuteScalar 和 ExecuteReader 方法的所有调用。
SkipExtraReads 此整数值与 CommandBehavior 值一起使用。设置后，不会在写入事务的 ExecuteScalar 方法中执行额外的 Read 调用。应格外小心地使用此值，因为它可能会导致异常行为。它仅供依赖旧的、不正确的行为的旧版应用程序使用。
