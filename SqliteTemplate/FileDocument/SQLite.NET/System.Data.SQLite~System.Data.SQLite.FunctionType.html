public enum FunctionType

成员

会员名称 描述
Scalar 标量函数设计为可调用并立即返回结果。示例包括 ABS（）、Upper（）、Lower（） 等。
Aggregate 聚合函数旨在累积数据直到调用结束，然后返回从累积数据中收集的结果。示例包括 SUM（）、COUNT（）、AVG（） 等。
Collation 整理序列用于以自定义方式对文本数据进行排序，并显示在 ORDER BY 子句中。通常，ORDER BY 中的文本使用不区分大小写的直接比较函数进行排序。自定义排序序列可用于以用户定义的方式更改文本排序的行为。
Window 窗口函数旨在对一组特定的行应用 aggregate 和 ranking 函数。
