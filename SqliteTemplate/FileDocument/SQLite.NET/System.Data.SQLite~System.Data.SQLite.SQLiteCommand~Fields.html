The fields of the SQLiteCommand class are listed below. For a complete list of SQLiteCommand class members, see the SQLiteCommand Members topic.

Public Static Fields

ForceExtraReads This integer value is used with CommandBehavior values. When set, extra Read calls are performed within the ExecuteScalar methods for all transactions. This value should be used with extreme care because it can cause unusual behavior.
GlobalCommandBehaviors These are the extra command behavior flags that should be used for all calls into the ExecuteNonQuery, ExecuteScalar, and ExecuteReader methods.
SkipExtraReads This integer value is used with CommandBehavior values. When set, extra Read calls are not performed within the ExecuteScalar methods for write transactions. This value should be used with extreme care because it can cause unusual behavior. It is intended for use only by legacy applications that rely on the old, incorrect behavior.
