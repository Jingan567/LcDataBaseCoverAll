SQLiteConnection.CreateHandle 方法

创建并返回新的托管数据库连接句柄。此方法仅供 ISQLiteConnectionPool 接口的实现使用。理论上，它可以被其他类使用;但是，不支持该用法。

public static object CreateHandle(
   IntPtr nativeHandle
);

参数

nativeHandle
这必须是 SQLite 核心库返回的本机数据库连接句柄，并且它必须在调用方法的整个持续时间内保持有效和打开。

返回值

新的托管数据库连接句柄，如果无法创建，则为 null。
