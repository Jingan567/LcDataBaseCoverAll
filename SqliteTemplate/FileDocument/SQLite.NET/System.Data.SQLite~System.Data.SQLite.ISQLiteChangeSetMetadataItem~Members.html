ISQLiteChangeSetMetadataItem 成员

ISQLiteChangeSetMetadataItem 概述

公共实例属性

Indirect 如果此更改被视为间接更改（即，就像它们是通过触发器或外键作进行的）时，则为非零。
NumberOfColumns 受此更改影响的列数。此值可用于确定可与此接口的 GetOldValue、GetNewValue 和 GetConflictValue 方法一起使用的最高有效列索引。它将是此值减 1。
NumberOfForeignKeyConflicts 当冲突类型为 ForeignKey 时，只能从 SessionConflictCallback 委托中调用此方法。它返回目标数据库中已知外键冲突的总数。
OperationCode 这将包含值 Insert、Update 或 Delete，对应于此项表示的总体更改类型。
PrimaryKeyColumns此数组包含表中与此更改关联的每列的布尔值。如果列不是主键的一部分，则该元素将为零;否则，它将为非零。
TableName 更改的表的名称。

公共实例方法

GetConflictValue 查询并返回此更改的给定列的冲突值。仅当冲突类型为 Data 或 Conflict 时，才能从 SessionConflictCallback 委托中调用此方法。
GetNewValue查询并返回此更改的给定列的更新值。仅当 OperationCode 的值为 Insert 或 Update 时，才能调用此方法。
GetOldValue查询并返回此更改的给定列的原始值。仅当 OperationCode 的值为 Update 或 Delete 时，才能调用此方法。
