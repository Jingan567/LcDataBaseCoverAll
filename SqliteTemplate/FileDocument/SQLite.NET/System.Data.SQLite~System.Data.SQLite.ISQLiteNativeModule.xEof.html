ISQLiteNativeModule.xEof 方法

int (*xEof)(sqlite3_vtab_cursor*);

如果指定的游标当前指向有效的数据行，则 xEof 方法必须返回 false（零），否则返回 true（非零）。此方法在每次 xFilter 和 xNext 调用后立即由 SQL 引擎调用。

每个虚拟表实现都需要 xEof 方法。

int xEof(
   IntPtr pCursor
);

Parameters

pCursor
指向 sqlite3_vtab_cursor 派生结构的本机指针。

返回值

如果没有更多行可用，则为非零;否则为 0。
