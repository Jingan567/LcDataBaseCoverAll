SQLiteConnection.Cancel 方法

此方法会导致任何待处理的数据库作中止并尽早返回。调用此例程通常是为了响应用户作，例如按 “Cancel” 或 Ctrl-C，其中用户希望立即停止长查询作。从任何线程调用此例程都是安全的。但是，使用已关闭或可能在此方法返回之前关闭的数据库连接来调用此例程是不安全的。
