SQLiteCommand 属性

下面列出了 SQLiteCommand 类的属性。有关 SQLiteCommand 类成员的完整列表，请参阅 SQLiteCommand 成员主题。

公共实例属性

CommandText 与命令关联的 SQL 命令文本
CommandTimeout 在出错之前等待连接变为可用的时间
CommandType 命令的类型。SQLite 仅支持 CommandType.Text
Connection重载。与此命令关联的连接
Container (inherited from Component)获取包含 Component 的 IContainer。
DesignTimeVisible 确定命令在设计时是否可见。默认为 True。
MaximumSleepTime 重试调用以准备或单步执行当前命令时休眠的最长时间。
Parameters重载。返回给定命令的 SQLiteParameterCollection
Site (inherited from Component)获取或设置 Component 的 ISite。
Transaction重载。与此命令关联的事务。SQLite 仅支持每个连接一个事务，因此此属性转发到命令的底层连接。
UpdatedRowSource 设置 SQLiteCommandBuilder 用于确定如何更新 DataTable 中插入或更新的行的方法。

受保护实例属性

CanRaiseEvents (inherited from Component)获取一个值，该值指示组件是否可以引发事件。
DbConnection 转发到本地 Connection 属性
DbParameterCollection 转发到本地 Parameters 属性
DbTransaction 转发到本地 Transaction 属性
DesignMode (inherited from Component)获取一个值，该值指示 Component 当前是否处于设计模式。
Events (inherited from Component)获取附加到此 Component 的事件处理程序的列表。
