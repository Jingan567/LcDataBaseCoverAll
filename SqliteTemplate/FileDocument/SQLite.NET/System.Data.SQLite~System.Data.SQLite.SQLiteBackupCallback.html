SQLiteBackupCallback 委托

在每个备份步骤之间引发。

public delegate bool SQLiteBackupCallback(
   SQLiteConnection source,
   string sourceName,
   SQLiteConnection destination,
   string destinationName,
   int pages,
   int remainingPages,
   int totalPages,
   bool retry
);

Parameters

source
源数据库连接。
sourceName
源数据库名称。
destination
目标数据库连接。
destinationName
目标数据库名称。
pages
每个步骤复制的页数.
remainingPages
要复制的剩余页数。
totalPages
源数据库中的总页数。
retry
如果由于数据库锁定问题而需要重试作，请设置为 true;否则，设置为 false。

返回值

如果 True 继续备份过程，则为 false 停止备份过程，回滚到目前为止所做的任何更改。

要求

命名空间： System.Data.SQLite

集会： System.Data.SQLite（System.Data.SQLite.dll）
