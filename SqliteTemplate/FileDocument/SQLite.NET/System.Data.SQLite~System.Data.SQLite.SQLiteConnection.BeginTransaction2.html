Parameters

deferredLock
当 TRUE 为 TRUE 时，SQLite 会延迟获取写入锁，直到请求写入作。当 FALSE 为 FALSE 时，将立即获得 writelock。默认值为 false，但在多线程多写入器环境中，可以选择立即锁定数据库以避免任何可能的写入器死锁。

返回值

返回 SQLiteTransaction 对象。
