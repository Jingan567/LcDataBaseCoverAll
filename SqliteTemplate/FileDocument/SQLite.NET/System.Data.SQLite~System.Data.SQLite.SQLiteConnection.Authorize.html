SQLiteConnection.Authorize 事件

每当 SQLite 在查询准备期间遇到授权方涵盖的作时，都会引发此事件。更改 ReturnCode 属性的值将确定是允许、忽略还是拒绝特定作。在事件的整个持续时间内，被调用的代码不得直接或间接修改关联的 connection 和 statement 对象。
