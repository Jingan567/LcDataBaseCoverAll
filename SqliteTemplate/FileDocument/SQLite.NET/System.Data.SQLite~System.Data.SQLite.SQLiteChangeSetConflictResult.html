SQLiteChangeSetConflictResult 枚举

此枚举类型表示用户定义的冲突解决回调的结果。

public enum SQLiteChangeSetConflictResult

成员

会员名称 描述
Omit 如果冲突回调返回此值，则不会执行任何特殊作。不会应用导致冲突的更改。更改的应用将继续进行下一次更改。
Replace 仅当冲突类型为 Data 或 Conflict 时，才能从冲突回调返回此值。如果不是这种情况，则到目前为止应用的任何更改都将回滚，并且对 Apply 的调用将引发错误代码为 Misuse 的 SQLiteException。如果为 Data conflict （数据冲突） 返回此值，则会更新或删除冲突的行，具体取决于更改的类型。如果为 Conflict 冲突返回此值，则会从数据库中删除冲突的行，并再次尝试应用更改。如果第二次尝试失败，则在继续之前将原始行恢复到数据库中。
Abort 如果返回此值，则到目前为止应用的任何更改都将回滚，并且对 Apply 的调用将引发错误代码为 Abort 的 SQLiteException。
