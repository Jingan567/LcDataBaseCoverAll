SQLiteBlob.Create（SQLiteConnection， String， String， String， Int64， Boolean） 方法

创建 SQLiteBlob 对象。这不适用于在没有 ROWID 的情况下创建的表。

public static SQLiteBlob Create(
   SQLiteConnection connection,
   string databaseName,
   string tableName,
   string columnName,
   long rowId,
   bool readOnly
);

Parameters

connection
打开 blob 对象时要使用的连接。
databaseName
包含 blob 对象的数据库的名称。
tableName
包含 blob 对象的表的名称。
columnName
包含 blob 对象的列的名称。
rowId
与所需 blob 对象关联的行的整数标识符。
readOnly
非零 打开 blob 对象进行只读访问。

返回值

如果发生错误，则新创建的 SQLiteBlob 实例 -OR- null 。
