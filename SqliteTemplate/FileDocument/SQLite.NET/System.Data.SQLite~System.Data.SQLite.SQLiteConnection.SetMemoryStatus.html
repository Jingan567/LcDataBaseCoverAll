SQLiteConnection.SetMemoryStatus Method

Sets the status of the memory usage tracking subsystem in the SQLite core library. By default, this is enabled. If this is disabled, memory usage tracking will not be performed. This is not really a per-connection value, it is global to the process.

public static SQLiteErrorCode SetMemoryStatus(
   bool value
);

Parameters

value
Non-zero to enable memory usage tracking, zero otherwise.

Return Value

A standard SQLite return code (i.e. zero for success and non-zero for failure).
