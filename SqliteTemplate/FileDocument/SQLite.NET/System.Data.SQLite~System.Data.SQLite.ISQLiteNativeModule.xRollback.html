ISQLiteNativeModule.xRollback 方法

int (*xRollback)(sqlite3_vtab *pVTab);

此方法会导致虚拟表事务回滚。此方法是可选的。sqlite3_module 的 xRollback 指针可能为 NULL。

对此方法的调用始终在对 xBegin 的先前调用之后。
