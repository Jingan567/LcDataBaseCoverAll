SQLiteCommand.SkipExtraReads Field

This integer value is used with CommandBehavior values. When set, extra Read calls are not performed within the ExecuteScalar methods for write transactions. This value should be used with extreme care because it can cause unusual behavior. It is intended for use only by legacy applications that rely on the old, incorrect behavior.

public const CommandBehavior SkipExtraReads = CommandBehavior.268435456;
