SQL 注释语法

comment-syntax：

注释不是 SQL 命令，但可以出现在 传递给 sqlite3_prepare_v2（） 和相关接口的 SQL 查询。 解析器将 Comments 视为空格。 注释可以从任何空格开始 ，包括跨多行的 Inside 表达式。

SQL 注释以两个连续的 “-” 字符 （ASCII 0x2d） 开头 并扩展到并包括下一个换行符 （ASCII 0x0a） 或直到 input 结束，以先到者为准。

C 样式注释开始 替换为 “/*” 并扩展到下一个 “*/” 字符对并包括下一个 “*/” 字符对 或直到 input 结束，以先到者为准。C 样式注释 可以跨越多行。

注释可以出现在任何可能出现空格的地方， 包括表达式内部和其他 SQL 语句的中间。 注释不嵌套。
