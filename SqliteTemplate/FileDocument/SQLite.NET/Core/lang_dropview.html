拖放视图

drop-view-stmt 的

DROP VIEW 语句将删除由 CREATE VIEW 语句创建的视图。视图定义已从数据库架构中删除，但 不会修改基础基表中的实际数据。

要删除的视图由 view-name 和 optional 标识 schema-name 指定为 DROP VIEW 语句的一部分。这 使用对象解析的标准过程解析引用。

如果找不到指定的视图，并且 IF EXISTS 子句不是 present，则表示这是一个错误。如果找不到指定的视图，并且 IF EXISTS 子句，则语句 是无作。
