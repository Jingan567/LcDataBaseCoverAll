安装 SQLite Visual Studio 设计时支持

支持 Visual Studio 查询设计器并允许您进行作 Visual Studio 中的 SQLite 数据库可以节省大量时间。虽然 支持尚未完全实现，肯定有足够的 让您忙碌。您可以创建数据库、设计和执行查询、 创建类型化数据集等等，所有这些都来自 Visual Studio。

安装说明

下载并运行其中一个安装包，然后选择“安装 Visual Studio 20XX 的设计器组件。选项。

Express Edition 限制

Visual Studio 设计时支持，适用于所有版本的 Visual Studio 2005/2008/2010/2012/2013/2015. 您可以将 SQLite 数据库添加到服务器列表中，设计 使用查询设计器进行查询，将表拖放到类型化 DataSet 上，等等。
由于 Visual Studio 许可限制，Express 版本不能 更长的时间得到支持。
