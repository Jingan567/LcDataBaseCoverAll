The methods of the ISQLiteChangeSetMetadataItem interface are listed below. For a complete list of ISQLiteChangeSetMetadataItem interface members, see the ISQLiteChangeSetMetadataItem Members topic.

Public Instance Methods

GetConflictValue Queries and returns the conflicting value of a given column for this change. This method may only be called from within a SessionConflictCallback delegate when the conflict type is Data or Conflict.
GetNewValue Queries and returns the updated value of a given column for this change. This method may only be called when the OperationCode has a value of Insert or Update.
GetOldValue Queries and returns the original value of a given column for this change. This method may only be called when the OperationCode has a value of Update or Delete.
