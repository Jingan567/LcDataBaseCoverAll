ISQLiteConnectionPool 方法

下面列出了 ISQLiteConnectionPool 接口的方法。有关 ISQLiteConnectionPool 接口成员的完整列表，请参阅 ISQLiteConnectionPool 成员主题。

公共实例方法

Add 将连接添加到与指定数据库文件名关联的对象的池中。
ClearAllPools 释放所有池连接。
ClearPool 释放与指定数据库文件名关联的所有池连接。
GetCounts 计算与指定文件名匹配的池条目数。
Remove 从与指定数据库文件名关联的连接池中删除连接，以便使用它与数据库进行交互。
