SessionConflictCallback 委托

当对数据库应用更改时发生冲突时，将调用此回调。

public delegate SQLiteChangeSetConflictResult SessionConflictCallback(
   object clientData,
   SQLiteChangeSetConflictType type,
   ISQLiteChangeSetMetadataItem item
);

Parameters

clientData
最初传递给 Apply 方法的可选应用程序定义的上下文数据。此值可能为 null。
type
此冲突的类型。
item
与此冲突关联的 ISQLiteChangeSetMetadataItem 对象。此值不能为 null;但是，只有适用于 Conflict 类型的属性才可用。有关此内容的更多信息，请参阅可用 SQLiteChangeSetConflictType 值的描述。

返回值

一个 SQLiteChangeSetConflictResult 值，该值指示为解决冲突而要采取的作。从此回调引发异常将导致未定义的行为。
