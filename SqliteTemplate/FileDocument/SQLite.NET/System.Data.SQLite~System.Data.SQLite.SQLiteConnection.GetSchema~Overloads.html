SQLiteConnection.GetSchema Method

The following commands are used to extract schema information out of the database. Valid schema types are:

  • MetaDataCollections
  • DataSourceInformation
  • Catalogs
  • Columns
  • ForeignKeys
  • Indexes
  • IndexColumns
  • Tables
  • Views
  • ViewColumns
