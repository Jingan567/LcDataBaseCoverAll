SQLiteConnection.Progress Event

This event is raised periodically during long running queries. Changing the value of the ReturnCode property will determine if the operation in progress will continue or be interrupted. For the entire duration of the event, the associated connection and statement objects must not be modified, either directly or indirectly, by the called code.
