ISQLiteNativeModule.xRowId 方法

int (*xRowid)(sqlite3_vtab_cursor *pCur, sqlite_int64 *pRowid);

成功调用此方法将导致 *pRowid 填充虚拟表游标 pCur 当前指向的行的 rowid。此方法在成功时返回 SQLITE_OK。失败时，它会返回相应的错误代码。

每个虚拟表实现都需要 xRowid 方法。

SQLiteErrorCode xRowId(
   IntPtr pCursor,
   ref long rowId
);

Parameters

pCursor
指向 sqlite3_vtab_cursor 派生结构的本机指针。
rowId
成功后，必须修改此参数以包含指定游标的当前行的唯一整数行标识符。

返回值

标准 SQLite 返回代码。
