SQLiteBlob 概述

公共静态方法

Create重载。创建 SQLiteBlob 对象。这不适用于在没有 rowid 的情况下创建的表-或者-如果查询不包括 “rowid” 列或其别名之一-或者-如果 SQLiteDataReader 不是使用 KeyInfo 标志创建的。
