SQLiteCommand.Execute（String， SQLiteExecuteType， String， Object） 方法

此方法创建新连接，使用给定的执行类型执行查询，关闭连接并返回结果。如果连接字符串为 null，则将使用临时内存数据库连接。

public static object Execute(
   string commandText,
   SQLiteExecuteType executeType,
   string connectionString,
   params object[] args
);

Parameters

commandText
要执行的命令的文本。
executeType
命令的执行类型。这用于确定要调用命令对象的哪个方法，然后确定返回的结果类型（如果有）。
connectionString
要打开、使用和关闭的数据库的连接字符串。如果此参数为 null，则将使用临时内存数据库。
args
构建要执行的命令对象时要使用的 SQL 参数值（如果有）。

返回值

查询结果 -OR- null 如果给定执行类型未生成任何结果。
