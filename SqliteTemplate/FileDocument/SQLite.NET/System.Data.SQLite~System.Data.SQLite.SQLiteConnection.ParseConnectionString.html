SQLite ADO.NET Provider

SQLiteConnection.ParseConnectionString 方法

使用自定义连接字符串解析器将连接字符串解析为组件部分。如果连接字符串的语法不正确，可能会引发异常。

public static SortedList<string, string> ParseConnectionString(
   SQLiteConnection connection,
   string connectionString,
   bool parseViaFramework,
   bool allowNameOnly,
   bool strict
);

Parameters

connection
将使用已分析连接字符串的连接。
connectionString
要解析的连接字符串。
parseViaFramework
非零，以使用框架本身提供的算法解析连接字符串。这在 .NET Compact Framework 上运行时不适用。
allowNameOnly
如果允许名称不带值，则为非零。
strict
非零值表示，如果任何连接字符串值不是 String 类型，则引发异常。这在 .NET Compact Framework 上运行时不适用。

返回值

与连接字符串中指定的参数对应的键/值对列表。
