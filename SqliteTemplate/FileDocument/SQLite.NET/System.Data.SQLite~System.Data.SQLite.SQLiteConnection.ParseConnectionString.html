SQLiteConnection.ParseConnectionString Method

Parses a connection string into component parts using the custom connection string parser. An exception may be thrown if the syntax of the connection string is incorrect.

public static SortedList<string, string> ParseConnectionString(
   SQLiteConnection connection,
   string connectionString,
   bool parseViaFramework,
   bool allowNameOnly,
   bool strict
);

Parameters

connection
The connection that will be using the parsed connection string.
connectionString
The connection string to parse.
parseViaFramework
Non-zero to parse the connection string using the algorithm provided by the framework itself. This is not applicable when running on the .NET Compact Framework.
allowNameOnly
Non-zero if names are allowed without values.
strict
Non-zero to throw an exception if any connection string values are not of the String type. This is not applicable when running on the .NET Compact Framework.

Return Value

The list of key/value pairs corresponding to the parameters specified within the connection string.
