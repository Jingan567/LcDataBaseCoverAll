SQLiteConnection.SetAvRetry Method

Queries or modifies the number of retries or the retry interval (in milliseconds) for certain I/O operations that may fail due to anti-virus software.

public SQLiteErrorCode SetAvRetry(
   ref int count,
   ref int interval
);

Parameters

count
The number of times to retry the I/O operation. A negative value will cause the current count to be queried and replace that negative value.
interval
The number of milliseconds to wait before retrying the I/O operation. This number is multiplied by the number of retry attempts so far to come up with the final number of milliseconds to wait. A negative value will cause the current interval to be queried and replace that negative value.

Return Value

Zero for success, non-zero for error.
