SQLiteBindValueCallback 委托

这表示为响应将参数绑定到命令的请求而调用的方法。如果引发异常，则会导致参数绑定作失败 -AND- 它将继续展开调用堆栈。

public delegate void SQLiteBindValueCallback(
   SQLiteConvert convert,
   SQLiteCommand command,
   SQLiteConnectionFlags flags,
   SQLiteParameter parameter,
   string typeName,
   int index,
   object userData,
   out bool complete
);

Parameters

convert
正在使用的 SQLiteConvert 实例。
command
正在使用的 SQLiteCommand 实例。
flags
与正在使用的 SQLiteConnection 实例关联的标志。
parameter
绑定到命令的 SQLiteParameter 实例。
typeName
与此回调关联的数据库类型名称。
index
绑定到命令的参数的序号。
userData
注册此回调时最初使用的数据。
complete
如果应跳过参数绑定调用的默认处理（即根本不应绑定参数），则为非零。将其设置为非零时应格外小心。

要求

命名空间： System.Data.SQLite

集会： System.Data.SQLite（System.Data.SQLite.dll）
