ISQLiteChangeSetMetadataItem 属性

下面列出了 ISQLiteChangeSetMetadataItem 接口的属性。有关 ISQLiteChangeSetMetadataItem 接口成员的完整列表，请参阅 ISQLiteChangeSetMetadataItem 成员主题。

公共实例属性

Indirect 如果此更改被视为间接更改（即，就像它们是通过触发器或外键作进行的）时，则为非零。
NumberOfColumns 受此更改影响的列数。此值可用于确定可与此接口的 GetOldValue、GetNewValue 和 GetConflictValue 方法一起使用的最高有效列索引。它将是此值减 1。
NumberOfForeignKeyConflicts 当冲突类型为 ForeignKey 时，只能从 SessionConflictCallback 委托中调用此方法。它返回目标 Databas 中已知外键冲突的总数e.
OperationCode 这将包含值 Insert、Update 或 Delete，对应于此项表示的总体更改类型。
PrimaryKeyColumns 此数组包含表中与此更改关联的每列的布尔值。如果列不是主键的一部分，则该元素将为零;否则，它将为非零。
TableName 更改的表的名称。
