ISQLiteChangeSetMetadataItem Properties

The properties of the ISQLiteChangeSetMetadataItem interface are listed below. For a complete list of ISQLiteChangeSetMetadataItem interface members, see the ISQLiteChangeSetMetadataItem Members topic.

Public Instance Properties

Indirect Non-zero if this change is considered to be indirect (i.e. as though they were made via a trigger or foreign key action).
NumberOfColumns The number of columns impacted by this change. This value can be used to determine the highest valid column index that may be used with the GetOldValue, GetNewValue, and GetConflictValue methods of this interface. It will be this value minus one.
NumberOfForeignKeyConflicts This method may only be called from within a SessionConflictCallback delegate when the conflict type is ForeignKey. It returns the total number of known foreign key violations in the destination database.
OperationCode This will contain the value Insert, Update, or Delete, corresponding to the overall type of change this item represents.
PrimaryKeyColumns This array contains a Boolean for each column in the table associated with this change. The element will be zero if the column is not part of the primary key; otherwise, it will be non-zero.
TableName The name of the table the change was made to.
