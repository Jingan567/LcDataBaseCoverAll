ISQLiteManagedModule 接口

此接口表示用托管代码编写的虚拟表实现。

有关此类型的所有成员的列表，请参阅 ISQLiteManagedModule Members 。

public interface ISQLiteManagedModule

实现 ISQLiteManagedModule 的类型

类型 描述
SQLiteModuleEnumerable(T) 此类实现一个虚拟表模块，该模块将 IEnumerable'1 对象实例公开为只读虚拟表。它不是密封的，可以用作包装 IEnumerable'1 对象实例的任何用户定义的虚拟表类的基类。
SQLiteModule 此类表示托管虚拟表模块实现。它不是密封的，必须用作在托管代码中实现的任何用户定义的虚拟表模块类的基类。
SQLiteModuleCommon 此类包含一些可能对其他虚拟 table 类有用的虚拟方法。它明确地不实现任何 ISQLiteManagedModule 接口方法。
SQLiteModuleEnumerable 此类实现一个虚拟表模块，该模块将 IEnumerable 对象实例公开为只读虚拟表。它不是密封的，可以用作包装 IEnumerable 对象实例的任何用户定义的虚拟表类的基类。下面的简短示例显示了它用于将字符串数组视为表数据源： 
   public static class Sample
   {
     public static void Main()
     {
       using (SQLiteConnection connection = new SQLiteConnection(
           "Data Source=:memory:;"))
       {
         connection.Open();

         connection.CreateModule(new SQLiteModuleEnumerable(
           "sampleModule", new string[] { "one", "two", "three" }));

         using (SQLiteCommand command = connection.CreateCommand())
         {
           command.CommandText =
               "CREATE VIRTUAL TABLE t1 USING sampleModule;";

           command.ExecuteNonQuery();
         }

         using (SQLiteCommand command = connection.CreateCommand())
         {
           command.CommandText = "SELECT * FROM t1;";

           using (SQLiteDataReader dataReader = command.ExecuteReader())
           {
             while (dataReader.Read())
               Console.WriteLine(dataReader[0].ToString());
           }
         }

         connection.Close();
       }
     }
   }
SQLiteModuleNoop 此类通过为所有 ISQLiteManagedModule 接口方法提供“空”实现来实现一个虚拟表模块，该模块不执行任何作。这些“空”方法实现返回的结果代码可以通过使用和/或重写派生类中的 GetDefaultResultCode、ResultCodeToEofResult、ResultCodeToFindFunctionResult、GetMethodResultCode 和 SetMethodResultCode 方法，在每个方法的基础上进行控制。
