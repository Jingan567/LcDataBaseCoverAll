SQLiteVirtualTableCursorEnumerator(T) Class

This class represents a virtual table cursor to be used with the SQLiteModuleEnumerable class. It is not sealed and may be used as the base class for any user-defined virtual table cursor class that wraps an IEnumerator`1 object instance.
