ISQLiteChangeSet 概述

公共实例方法

Apply重载。尝试将此实例中的更改集应用于关联的数据库。
CombineWith 此方法将指定的更改集与此实例中包含的更改组合在一起。
Invert 此方法 “反转” 此实例中的更改集。将一组反转的更改应用于数据库将反转应用未反转更改的效果。具体来说： <ul><li> 每个 DELETE 更改都更改为 INSERT，</li><li> 每个 INSERT 更改都更改为 DELETE，</li><li> 对于每个 UPDATE 更改，都会交换 old.* 和 new.* 值。</li></ul> 此方法不会更改更改在更改集中的显示顺序。它只是颠倒了每个单独变化的意义。
