SQLiteModuleEnumerable(T) Class

This class implements a virtual table module that exposes an IEnumerable`1 object instance as a read-only virtual table. It is not sealed and may be used as the base class for any user-defined virtual table class that wraps an IEnumerable`1 object instance.
