SQLiteConnection.BindFunction（SQLiteFunctionAttribute， Delegate， Delegate） 方法

尝试将指定的 SQLiteFunction 对象实例绑定到此连接。

public void BindFunction(
   SQLiteFunctionAttribute functionAttribute,
   Delegate callback1,
   Delegate callback2
);

Parameters

functionAttribute
包含要绑定的函数的元数据的 SQLiteFunctionAttribute 对象实例。
callback1
一个 Delegate 对象实例，可帮助实现要绑定的函数。对于标量函数，这对应于 SQLiteInvokeDelegate 类型。对于聚合函数，这对应于 SQLiteStepDelegate 类型。对于排序规则函数，这对应于 SQLiteCompareDelegate 类型。
callback2
一个 Delegate 对象实例，可帮助实现要绑定的函数。对于聚合函数，这对应于 SQLiteFinalDelegate 类型。对于其他回调类型，它不被使用，并且必须为 null。
