SessionTableFilterCallback 委托

当必须确定是否应跟踪对特定表的更改 - OR- 应用时，将调用此回调。对于已附加到 ISQLiteSession 的表，不会调用它。

public delegate bool SessionTableFilterCallback(
   object clientData,
   string name
);

Parameters

clientData
最初传递给 SetTableFilter 或 Apply 方法的可选应用程序定义的上下文数据。此值可能为 null。
name
表的名称。

返回值

如果应考虑对表进行更改，则为非零;否则为 0。从此回调引发异常将导致未定义的行为。

要求

命名空间： System.Data.SQLite

集会： System.Data.SQLite（System.Data.SQLite.dll）
