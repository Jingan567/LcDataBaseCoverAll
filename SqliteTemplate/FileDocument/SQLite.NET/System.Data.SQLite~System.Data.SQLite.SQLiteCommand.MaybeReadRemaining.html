SQLiteCommand.MaybeReadRemaining 方法

检查是否需要对 Read 方法进行额外调用。如果是这样，它会尝试执行这些作。如果没有，则什么都不做。

public int MaybeReadRemaining(
   SQLiteDataReader reader,
   CommandBehavior behavior
);

Parameters

reader
从 ExecuteReader 方法之一接收的数据读取器实例。
behavior
传递到其中一个查询执行方法的原始命令行为标志。

返回值

对 Read 执行的额外调用数 -OR- 负数 1 以指示它们未启用。
