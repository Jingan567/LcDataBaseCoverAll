SQLiteConnection.GetCriticalHandle Method

Attempts to obtain and return the underlying CriticalHandle derived object associated with this connection. This method should only be used by the thread that created this connection; otherwise, the results are undefined. 

WARNING: This method is not officially supported for external callers and
         should be considered "experimental", even though it is "public".

public object GetCriticalHandle();

Return Value

The underlying CriticalHandle derived object associated with this connection -OR- null if it is unavailable.
