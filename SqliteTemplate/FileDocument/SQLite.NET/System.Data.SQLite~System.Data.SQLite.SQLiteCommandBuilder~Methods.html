CreateObjRef (inherited from MarshalByRefObject)创建一个对象，该对象包含生成用于与远程对象通信的代理所需的所有相关信息。

Equals (inherited from Object)确定指定的 Object 是否等于当前 Object。

QuoteIdentifier 将标识符括起来
RefreshSchema (inherited from DbCommandBuilder)清除与此 DbCommandBuilder 关联的命令。
